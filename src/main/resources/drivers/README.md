# JDBC Drivers Directory

Place your JDBC driver JAR files here.

## Common JDBC drivers:

### MySQL / MariaDB
- MySQL: https://dev.mysql.com/downloads/connector/j/
- MariaDB: https://mariadb.com/downloads/connectors/

### PostgreSQL
- https://jdbc.postgresql.org/download.html

### SQL Server
- https://learn.microsoft.com/en-us/sql/connect/jdbc/download-microsoft-jdbc-driver-for-sql-server

### Oracle
- https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html

### SQLite
- https://github.com/xerial/sqlite-jdbc

The application will automatically load all JAR files found in this directory at startup.
